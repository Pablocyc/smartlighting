Smart Lighting

Smart lighting es el término que utilizamos para referirnos a un sistema inteligente de iluminación. El objetivo no es otro que el de propiciar un uso cada vez más eficiente de los recursos reduciendo el coste final del sistema manteniendo o mejorando el nivel de iluminación.

Quienes Somos

La Comunidad Open Source Hardware es un grupo de personas con el interés común de compartir conocimiento, estrategias para diseñar, investigar y construir diferentes tipos de dispositivos electrónicos basados en el uso del software y hardware libre.
